Akara Podcast application has been created using:
- Java version "17.0.5"
- JavaFX version 19.0.0
- Build system: Maven
- IntelliJ IDEA 2022.3.1 (Ultimate Edition)

**Clone the source code**
```sh
$ git  clone  https://github.com/akara-podcast/akara-podcast.git
```
